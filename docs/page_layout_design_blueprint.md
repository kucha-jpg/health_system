# 页面布局与功能蓝图（角色化）

## 1. 设计目标
- 信息更丰富：每个页面都包含“当前状态 + 主操作 + 数据区 + 结果反馈”。
- 页面不臃肿：一个页面只保留一个主任务，次要操作收纳到筛选区或二级入口。
- 视觉更干净：统一间距、控件尺寸、卡片层级、表格风格与提示语语气。
- 角色更清晰：管理员看治理，医生看处置，患者看自我管理。

## 2. 通用页面骨架（所有业务页面）
- 顶部：页面标题 + 一句副标题（任务导向）。
- 导览条：角色标识、当前页面、角色焦点提示。
- 筛选与操作区：查询、重置、导出、批量操作，按主次分两行。
- 主数据区：卡片、图表或表格，只保留一个视觉主角。
- 底部：分页、空状态、错误提示。

## 3. 角色与页面设计

### 3.1 管理员（ADMIN）

#### 工作首页
- 内容：角色焦点、三张快捷入口卡（账号管理/系统监控/反馈消息）、最新公告。
- 功能：快速跳转核心治理页面，查看公告详情。

#### 账号管理
- 内容：筛选区（关键词/角色/状态）、统计卡（当前页/启用/禁用）、账号表格。
- 功能：新增、编辑、启停、分页查询。

#### 系统公告
- 内容：筛选区（关键词/状态/投放对象）、公告列表、富文本编辑弹窗。
- 功能：新增/编辑/删除、富文本预览、按角色投放。

#### 预警规则
- 内容：规则列表 + 指标类型列表，双区块并列或上下结构。
- 功能：阈值维护、启停控制、指标维护。

#### 角色权限
- 内容：角色表格（角色名、权限字符串、状态）。
- 功能：编辑权限、快速回滚建议（可后续增加）。

#### 系统监控
- 内容：核心统计卡、趋势图、分布图、Top列表。
- 功能：时间维度观察、导出概览。

#### 操作日志
- 内容：筛选区（关键词/角色/结果/时间范围）、阈值设置、日志表格。
- 功能：CSV导出、慢操作筛查、批量行为筛查。

#### 反馈消息
- 内容：统计卡、近7天趋势、筛选区、反馈表格、回复弹窗。
- 功能：单条回复、批量状态流转、按筛选跨页批量、导出。

### 3.2 医生（DOCTOR）

#### 工作首页
- 内容：角色焦点、快捷入口卡（医生工作台/群组管理/反馈通道）、最新公告。
- 功能：优先进入高风险处理链路。

#### 医生工作台（预警）
- 内容：筛选区（风险级别/最低分/排序）、统计卡、预警列表。
- 功能：闭环处理、一键闭环、分页处理。

#### 群组管理
- 内容：群组列表、成员查看弹窗、协作医生弹窗。
- 功能：新建群组、添加患者、添加医生、跳转患者洞察。

#### 患者洞察
- 内容：患者基础信息、档案区、趋势图、预警分布、明细表。
- 功能：按指标与时间过滤、按备注/状态过滤、图表联动。

#### 反馈通道
- 内容：反馈提交区、历史反馈表。
- 功能：提交反馈、按时间范围筛选、查看回复。

### 3.3 患者（PATIENT）

#### 工作首页
- 内容：角色焦点、快捷入口卡（健康上报/预警详情/周报月报）、最新公告。
- 功能：快速进入每日核心任务。

#### 个人档案
- 内容：基础信息表单（病史/用药/过敏）。
- 功能：保存档案、后续可加入完整性提示。

#### 健康上报
- 内容：指标选择、数值输入、时间、备注。
- 功能：数据校验、提交上报。

#### 历史数据
- 内容：筛选区、统计卡、趋势图、历史表格。
- 功能：筛选、编辑、删除、分页。

#### 预警详情
- 内容：状态筛选、预警列表、分页。
- 功能：按状态查看处理进度。

#### 个性化阈值
- 内容：指标切换、阈值表单、配置表格。
- 功能：保存阈值、恢复推荐值、跳转预警结果。

#### 周报月报
- 内容：周/月切换、筛选区、风险趋势图、指标分布图、最近上报表。
- 功能：导出报表、图表类型切换。

#### 反馈通道
- 内容：反馈提交区、反馈历史与回复。
- 功能：提交、筛选、查看处理结果。

## 4. 视觉规范（已落地方向）
- 统一卡片：弱阴影、半透明浅底、圆角一致。
- 统一筛选区：固定宽度类 + 自适应折行。
- 统一表格：表头加重、背景降噪、空状态统一插画块。
- 统一按钮：主次色明确，主要动作突出，次要动作文字化。
- 统一提示：软提示条只写状态，不写长段指导文。

## 5. 后续增强建议（不影响当前版本）
- 每页增加“最近操作记录”抽屉，提升可追溯性。
- 关键流程加入“完成度”提示（如档案完整度、规则覆盖度）。
- 为管理员和医生增加“今日待办”聚合卡片（可配置）。
