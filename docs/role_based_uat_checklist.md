# 角色实机验收脚本（UAT）

日期：2026-04-07
适用版本：前端全站重设计 + 主题切换

## 使用说明

1. 先登录对应角色账号。
2. 顶栏切换三套主题（森氧绿/海岸蓝/日光暖）后再执行每条用例。
3. 每条用例记录“通过/不通过”，不通过时写明现象和页面。

记录格式建议：

- 结果：通过 / 不通过
- 备注：问题现象、复现步骤、截图文件名
