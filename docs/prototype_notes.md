# 原型说明与页面流转（附录材料）

日期：2026-04-07

## 1. 角色页面流转

### 1.1 患者端

- 登录 -> 工作首页 -> 健康上报 -> 历史数据 -> 周报/月报 -> 预警详情 -> 反馈通道

### 1.2 医生端

- 登录 -> 工作首页 -> 预警工作台 -> 群组管理 -> 患者洞察 -> 反馈通道

### 1.3 管理员端

- 登录 -> 工作首页 -> 账号管理 -> 角色权限 -> 预警规则 -> 系统监控 -> 操作日志 -> 反馈消息

## 2. 关键交互点

1. 医生预警支持风险筛选与一键闭环。
2. 患者报表支持指标筛选与 CSV 导出。
3. 管理员监控支持时间/群组/用户/指标多维分析与导出。
4. 患者支持个性化阈值配置并实时生效。

## 3. 原型评审建议

1. 先演示“患者上报 -> 医生预警处理 -> 管理员监控”的完整闭环。
2. 再演示“个性化阈值 + 趋势预测预警”的智能化能力。
3. 最后演示“权限隔离 + 审计日志 + 错误码规范”的安全能力。
